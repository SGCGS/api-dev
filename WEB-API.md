# 鉴权
|权限等级|身份|
|---|---|
|0|游客和被封用户|
|1|已登录用户|
|2|管理员|

## 登录
### ManageBac 验证
|请求路径|请求方式|
|---|---|
|`/login/managebac`|GET|

参数：
|名称|注释|
|---|---|
|username|ManageBac 用户名|
|password|ManageBac 密码|
|rt|reCaptcha 验证令牌|

返回：  
|状态|响应代码|类型|内容|
|---|---|---|---|
|成功|200|空|响应头设置 Cookie|
|失败|401|JSON|{"detail":"失败提示"}|
